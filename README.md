### How to run it on your local
1. Clone this repository:
   `git clone https://github.com/chenflim/sltr_fannylimin.git`

2. Run `npm install`
3. Run `node src/server.js`
4. To run the test, run:
   `npm test`
5. Run docker image:
  `cd docker`
  `docker-compose up -d`
